\# 🚀 Ram Cleaner - Free Edition (by AT COMPANY)



\*\*Ram Cleaner\*\* is a lightweight, powerful, and smart utility designed to optimize your Windows RAM usage in real-time. Developed by \*\*Ultra Code Space\*\*, it ensures your system stays fast and responsive without manual intervention.



\---



\## ✨ Key Features



\* \*\*⚡ Manual Clean:\*\* One-click deep memory optimization.

\* \*\*⏰ Scheduled Cleaning:\*\* Auto-clean every 5, 10, or 60 minutes.

\* \*\*📉 Usage-Based Trigger:\*\* Automatically cleans when RAM usage hits a specific percentage.

\* \*\*🌓 Soft Light \& Dark Modes:\*\* Eye-friendly UI with a sleek, modern aesthetic.

\* \*\*🚀 Run on Startup:\*\* Starts with Windows to keep your PC optimized from the get-go.

\* \*\*🔔 Smart Notifications:\*\* Get notified whenever the cleaning process is completed.

\* \*\*🔄 Auto-Updater:\*\* Stay up-to-date with the latest features and fixes automatically.
